# User Guide

## Getting Started

### Registration and Login

1. **Register a New Account**

   - Navigate to the registration page
   - Enter your name, email, and password
   - Click "Register" to create your account
   - You'll be automatically logged in

2. **Login**
   - Go to the login page
   - Enter your email and password
   - Click "Login"

### Dashboard

The dashboard provides an overview of your projects and tasks:

- **Statistics**: View total projects, tasks, completed tasks, and active projects
- **Recent Projects**: Quick access to your latest projects
- **Recent Tasks**: Quick access to your latest tasks

## Projects

### Creating a Project

1. Navigate to the "Projects" page
2. Click the "+ New Project" button
3. Fill in the project details:
   - **Name**: Required
   - **Description**: Optional
   - **Status**: Planning, Active, On Hold, or Completed
   - **Priority**: Low, Medium, High, or Urgent
4. Click "Create"

### Managing Projects

- **View Project**: Click on any project card to view details
- **Add Members**: In project detail, click "Add Member" to invite team members
- **Create Tasks**: Click "+ New Task" to add tasks to the project
- **View Tasks**: All project tasks are listed in the project detail page

## Tasks

### Creating a Task

1. Navigate to a project detail page
2. Click "+ New Task"
3. Fill in task details:
   - **Title**: Required
   - **Description**: Optional
   - **Status**: To Do, In Progress, Review, or Done
   - **Priority**: Low, Medium, High, or Urgent
   - **Assign To**: Select a team member (optional)
4. Click "Create"

### Managing Tasks

- **View All Tasks**: Go to the "Tasks" page to see all your tasks
- **Filter Tasks**: Use the status and priority filters on the Tasks page
- **View Task Details**: Click on any task to view full details
- **Edit Task**: Click "Edit" on the task detail page
- **Add Comments**: Click "+ Add Comment" to discuss the task
- **Update Status**: Change task status as work progresses
- **Delete Task**: Click "Delete" to remove a task (only if you have permission)

### Task Statuses

- **To Do**: Task not yet started
- **In Progress**: Task is being worked on
- **Review**: Task is ready for review
- **Done**: Task is completed

## Teams

### Creating a Team

1. Navigate to the "Teams" page
2. Click "+ New Team"
3. Enter team name and description
4. Click "Create"

### Managing Teams

- **Add Members**: Click "Add Member" in team detail page
- **View Projects**: See all projects associated with the team
- **View Members**: See all team members and their roles

## Real-time Features

TaskFlow includes real-time collaboration features:

- **Live Updates**: Changes to projects and tasks appear instantly for all team members
- **Notifications**: Real-time notifications when tasks are created, updated, or commented on
- **Collaboration**: Multiple users can work on the same project simultaneously

## Best Practices

1. **Organize with Projects**: Group related tasks into projects
2. **Use Teams**: Create teams for departments or groups
3. **Set Priorities**: Use priority levels to focus on important work
4. **Add Descriptions**: Provide clear descriptions for better understanding
5. **Assign Tasks**: Assign tasks to specific team members
6. **Use Comments**: Communicate about tasks using comments
7. **Update Status**: Keep task statuses up to date

## Keyboard Shortcuts

- **Navigation**: Use browser back/forward buttons
- **Search**: Use browser search (Ctrl+F / Cmd+F) to find content

## Troubleshooting

### Can't Login

- Verify your email and password are correct
- Make sure you've registered an account
- Clear browser cache and try again

### Not Seeing Updates

- Refresh the page
- Check your internet connection
- Ensure you're logged in

### Permission Errors

- Verify you're a member of the project/team
- Contact the project owner for access

## Support

For technical issues or questions, please contact the development team or refer to the API documentation.
